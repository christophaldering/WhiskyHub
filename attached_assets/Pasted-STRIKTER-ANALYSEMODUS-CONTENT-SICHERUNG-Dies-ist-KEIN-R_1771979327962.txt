STRIKTER ANALYSEMODUS – CONTENT-SICHERUNG

Dies ist KEIN Refactoring-Auftrag.
Dies ist KEIN Aufräumauftrag.
Dies ist KEIN Optimierungsauftrag.

Ziel:
Ausschließlich vollständige Transparenz und Sicherung bestehender Inhalte.

ABSOLUTE VERBOTE:
- Keine Datei löschen
- Keine Datei verschieben
- Keine Datei umbenennen
- Keine Route entfernen
- Keine Route ändern
- Keine Komponenten zusammenführen
- Keine Inhalte kürzen
- Keine Texte verändern
- Keine Inhalte als „deprecated“ markieren
- Keine Logik anfassen
- Keine Imports entfernen
- Keine Features deaktivieren
- Keine UI-Elemente entfernen
- Keine Navigation ändern

Falls eine vorgeschlagene Handlung auch nur indirekt eine dieser Aktionen erfordert:
STOPPEN und explizit melden.

--------------------------------------------------
AUFGABE
--------------------------------------------------

1) Identifiziere alle Seiten, Module oder Komponenten, die:
   - methodische Reflexionen enthalten
   - Hintergrundwissen zu Verkostung, Bewertung, Testen oder Messen enthalten
   - Artikel, Bibliotheken oder Forschungsinhalte darstellen
   - umfangreiche Textinhalte (>300 Wörter) enthalten
   - frühere Wissensbereiche repräsentieren

2) Identifiziere alle existierenden Routen im Router (Wouter),
   die derzeit NICHT über irgendeine Navigation erreichbar sind.

3) Für jede identifizierte Seite aus Punkt 1 und 2 ausgeben:

   - Dateipfad
   - Route
   - Rollen-Gating
   - Aktuelle Erreichbarkeit:
       • Hauptmenü
       • Untermenü
       • Kontextbutton
       • Redirect
       • Nur manuelle URL
   - Kurze inhaltliche Beschreibung (max 3 Sätze)

4) Andock-Vorschläge (nur Vorschläge, keine Umsetzung):
   - Maximal 2 Integrationsmöglichkeiten nennen
   - Oder markieren: "Keine sinnvolle Integration erkennbar"

5) Falls keine Integration sinnvoll erscheint:
   - Vorschlag: Sichtbarmachung im Admin-Bereich unter
     „Archivierte Inhalte / Legacy Content“
   - Aber KEINE Umsetzung durchführen.

--------------------------------------------------
ABSCHLUSS
--------------------------------------------------

Gib am Ende ausschließlich folgende strukturierte Zusammenfassung aus:

- Anzahl identifizierter Wissensinhalte
- Anzahl existierender, aber nicht navigierter Routen
- Anzahl potenziell verwaister Seiten
- Anzahl sinnvoll integrierbarer Inhalte
- Anzahl nur als Admin-Archiv geeigneter Inhalte
