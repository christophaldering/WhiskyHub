## Backend

### 7. `shared/schema.ts`
```bash
grep -n "balance" shared/schema.ts
```
Falls balance noch als Spalte vorhanden:
- Aus ratings-Tabelle entfernen
- Aus allen Zod-Insert/Select-Schemas entfernen
- Migration erstellen:
```sql
ALTER TABLE ratings DROP COLUMN IF EXISTS balance;
```
Dann: `npm run db:push`

### 8. `server/routes.ts`
```bash
grep -n "balance" server/routes.ts
```
Typische Fundstellen:
