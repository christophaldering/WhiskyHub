# Replit Agent Prompt — AI Import als primäres Feature

-----

## DESIGNPRINZIP

Ein Apple Designer würde niemals zwei gleichwertige Optionen nebeneinander stellen.
Er würde entscheiden: Was ist der beste Weg? → Das wird primär.
Was ist der Fallback? → Das wird sekundär, aber würdevoll.

AI Import ist der beste Weg. Manuell ist der Fallback.

-----

## ÄNDERUNG 1: WHISKY LINEUP SCREEN

### Aktueller Zustand (schlecht)

- “AI Import” kleiner Link mit Icon
- “WB # | 🌐 | Whisky name | Add” — gleich prominent daneben
- Kein visueller Unterschied, kein klarer primärer Weg

### Neuer Zustand

```
┌─────────────────────────────────────────┐
│  ✦  Lineup mit KI erstellen             │

│  [ Lineup importieren          →  ]     │
└─────────────────────────────────────────┘

oder manuell hinzufügen:
[ WB # ]  [ Whisky name        ] [ + ]
```

## ÄNDERUNG 2: QUICK-START KARTE (Tasting erstellen)

### Wo

In `LabsHostDashboard.tsx` — die Quick-Start Karte aus dem Master-Prompt (Stufe 3), direkt nach dem “Anzahl Whiskies” Feld.

### Neues Element: AI Lineup Hint

Unmittelbar unter dem Toggle “Blind Tasting”, vor dem “Erstellen” Button, eine kleine inline Hinweiszeile:

```
┌─────────────────────────────────────────┐
│  [Name des Abends          ]            │
│                                          │
│  Wie viele Whiskies?                    │
│  [ 3 ] [ 4 ] [ 5 ] [ 6 ] [ 8+ ]        │
│                                          │
│  [●] Blind Tasting                      │
│                                          │
│  ✦ Lineup nach dem Start mit KI         │  ← NEU
│    importieren — Foto genügt.           │  ← NEU
│                                          │
│  [ Tasting erstellen & Code generieren ]│
└─────────────────────────────────────────┘
```

**Design des Hinweises:**

```css
display: flex;
align-items: flex-start;
gap: 8px;
padding: 12px 16px;
background: rgba(201,151,43,0.06);
border: 1px solid rgba(201,151,43,0.15);
border-radius: 12px;
margin-bottom: 16px;
```

Sparkles Icon: 14px, #E8B84B
Text EN: “Add your lineup after creating — a photo is all you need.”
Text DE: “Lineup nach dem Erstellen per Foto importieren.”
Font: 13px, font-weight 300, color: rgba(240,230,211,0.6)

## ÄNDERUNG 3: EMPTY LINEUP STATE

Wenn WHISKIES (0) — also noch keine Whiskies im Lineup — zeige statt der leeren Liste + kleinem AI-Link:

Vollbreite Karte, zentriert:

```
        ✦
  No whiskies yet.
  
  [ Import with AI ]     [ Add manually ]
  ←── primary ───        ←── ghost ────
```

**“Import with AI” Button:** gold gradient, primary
**“Add manually” Button:** transparent, border ghost, kleiner
